
# ✈️ Travel Recommendation System
A dynamic web application that provides personalized travel recommendations based on user preferences and data from a custom API.
## 💻 Technologies
* HTML, Vanilla JavaScript, and CSS
* Custom Travel Recommendation API (JSON)
## 🚀 Features
* **Smart Suggestions:** Provides tailored travel destinations based on specific categories like beaches, temples, or countries.
* **Interactive UI:** A fully responsive, user-friendly interface designed for seamless navigation.
* **Destination Insights:** Displays high-quality images, detailed descriptions.
* **Instant Search:** Quickly filter and find destinations using a keywords such as `country`, `beach`, `temples`.
## 🧠 What I Learned
* **API Logic:** Managing asynchronous data flow using the **Fetch API** to retrieve information from a `local JSON data source`.
* **Responsive Layouts:** Implementing CSS flexbox to ensure the platform looks great on screens.
* **Dynamic Content Injection:** Using JavaScript to parse JSON data and dynamically create HTML cards for each destination.
* **Search Filtering:** Developing logic to match user input strings with data attributes to provide real-time results.
## ⚙️ The Process
The project started with structuring the data model in a `JSON format` to simulate a real-world API. I then built a clean, minimalist front-end using `HTML` and `CSS`. The core challenge was ensuring that the JavaScript logic could handle multiple categories and display the correct data `without reloading the page`. I focused heavily on the user experience, making sure the search results appeared instantly and the design remained consistent across all viewports.
## ⚡ Running the Project
1. **Clone** the repository to your local machine.
2. **Setup:** Ensure the `travel_recommendation_api.json` is in the project root folder.
3. **Launch:** Open `travel_recommendation.html` in any modern web browser.
4. **Interact:** Use the search bar to type categories like "country", "beach" or "temple" to see the action
5. **Tip:** Do not forget to scroll over the cards suggested upon search.
